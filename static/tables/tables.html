Reference Tables

This page provides an overview of the 'recognized' code tables. By 'recognized', we mean that the table may be explicitly referenced in a message through the S attribute of the cd element.

The site provides the labels associated with the codes for the tables that are KMEHR-specific, such as the technical tables or several application-specific codifications. For international codifications and national codifications from other authentic sources, it only provides the identification of the table as well as some examples.

Tables with ‘deprecated’ status are kept for backward compatibility purposes but should no longer be used. Tables with ‘draft’ status should be used for testing purposes only.

The versions of the tables are listed in a spreadsheet.
